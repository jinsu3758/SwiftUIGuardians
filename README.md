## Rule
- 주 2회 모임을 가지고, 각 모임 전에 2개의 영상을 미리 보고 온다.
  - 주 2회 모임은 화,목
  - 2개 영상이 너무 양이 많을 경우 미리 합의해서 1개로 줄이던지 한다.
- 코드를 항상 작성해서 오며, 그에 대한 리뷰와 회고를 한다.
  - **강의에 나온 코드**와 추가적으로 하고 싶은 코드(필수 x) 포함
- 이 규칙은 언제든지 변할 수 있다. (미리 합의하여)
